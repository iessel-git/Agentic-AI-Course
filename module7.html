Module 7: Build Your Own Agent

Step-by-step instructions to create a simple Agentic AI

Overview

Now that you understand the theory, it's time to put it into practice. In this module, you'll build a basic autonomous agent that can make decisions and perform actions based on user input.

Tools You’ll Need

  • Basic HTML/CSS for interface
  • JavaScript to drive logic
  • Optional: Supabase for memory or backend

Code Example


// A simple decision-making agent
function agent(input) {
  if (input.includes("weather")) {
    return "The weather is sunny.";
  } else if (input.includes("joke")) {
    return "Why don't robots take vacations? They recharge at home!";
  } else {
    return "I'm still learning! Try asking something else.";
  }
}

// Example use:
console.log(agent("Tell me a joke"));

Hands-on Task

Build an agent that responds to the following:

  • User says “hello” → agent says “Hi, I’m your assistant agent.”
  • User mentions “help” → agent offers 2 suggestions
  • Else → agent says it’s learning
